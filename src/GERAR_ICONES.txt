⚡ MÉTODO MAIS RÁPIDO:

1. Abra este arquivo no navegador:
   
   📁 public/GERAR_ICONES_PNG.html
   
   (Vá até a pasta "public" e clique duas vezes no arquivo)

2. Clique em "📦 Baixar Ambos os Ícones"

3. Mova os 2 arquivos para a pasta /public/:
   - icon-192.png
   - icon-512.png

4. Execute:
   
   🪟 Windows: REBUILD_COM_ICONES_NOVOS.bat
   🐧 Linux/Mac: REBUILD_COM_ICONES_NOVOS.sh

5. No Android Studio:
   Build → Clean → Rebuild → Build APK

6. Instale o APK no celular!
